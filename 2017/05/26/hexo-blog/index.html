使用coding+hexo搭建博客

本篇主要介绍使用腾讯coding平台，结合hexo搭建博客

1. 创建coding账号

  • 注册地址：https://coding.net/register
  • coding web ide：https://studio.dev.tencent.com/

2. 搭建本地hexo环境

  • 安装git和nodejs
  • 安装npm
npm set registry https://registry.npm.taobao.org/

npm install -g hexo-cli 
npm install hexo-deployer-git hexo-server --save

3. 配置hexo并生成博客

  • 新建两个项目，一个用于存放生成blog的代码仓库，一个用于真正展示blog的代码仓库。注意blog所在的代码仓库需要保持和coding.net注册名保持一致，如创建项目名MrEason.coding.me，这样当blog生成完毕后，可以直接通过mreason.coding.me网址访问blog。举例如下：
    • easonblog项目用于生成Blog
    • MrEason.coding.me项目用于展示Blog
  • 配置并生成blog
# 克隆
git clone https://git.dev.tencent.com/MrEason/easonblog.git
cd easonblog
# 初始化
hexo init
# 编辑_config.yml
vim _config.yml

## 内容如下：
title: EasonBlog
subtitle: Eason's Blog
description: Love MaMin
keywords:
author: Eason Sun
language: en-US
timezone: Asia/Shanghai

# 此处填写blog真正展示时使用的代码仓库，如https://git.dev.tencent.com/MrEason/MrEason.coding.me.git

deploy:
  type: git 
  repo: git@git.coding.net:xxxxxxx/xxxxxxxx.git
  branch: master
  • 本地浏览：hexo server
  • 访问地址：http://localhost:4000

若要提交生成blog的代码仓库，需要注意.gitignore里的内容

public/
.deploy*/
  • 上传代码：
git add -A
git commit -m "first commit"
git push origin master
  • 常用操作：
# 新建文章
hexo new "title"
# 清空样式
hexo clean
# 生成
hexo g
# 部署
hexo d
  • 打开pages服务：代码 –> Pages服务

4. 更换博客主题

hexo主题：https://hexo.io/themes/
可以在网上搜索相关主题，并将其下载到blog/themes下，注意如果需要上传生成blog的代码仓库，最好是在其他目录下git clone后，去除.git目录后拷贝到themes目录；
根据每个主题的特性，修改相应参数。

5. 同时更新GithubPages

  • 新建项目：<注册名>.github.io，如EasonAssassin.github.io

  • 注意项目必须是public

  • settings里开启：

  • _config.yml添加如下配置

deploy:
  type: git
  repo:
    coding: git@git.dev.tencent.com:MrEason/MrEason.coding.me.git
    github: git@github.com:EasonAssassin/EasonAssassin.github.io.git
  branch: master
  message: blog update

6. 插入图片

  • 绝对路径
# 新建images目录
mkdir -p source/images/
# 在source/images里放置图片

hexo new hexo-blog
# 上述操作会在source/_posts/目录中生成hexo-blog.md和hexo-blog目录
